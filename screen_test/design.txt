design

表
	T1. ニコられ数とコメントスタイルの対応		
		ニコられ数 : スタイル
			0 : なし
			1以上3未満 : 太字
			3以上10未満 : 太字・文字色橙
			10以上 : 太字・文字色赤

仕様
	component
		C1. ニコるツールチップ
			image : nicoru_tooltip.png
			parts :
				1. ニコるくん画像
				2. ニコられ数 : コメントのニコられ数。最大 3 桁
			layout : 左から 1, 2
			place : コメントの左側
		C2. 総ニコられ数
			image : total_nicorare.png
			parts :
				1. スラッシュ
				2. ニコるくん画像
				3. 総ニコられ数 : 視聴中の動画のニコられ数の合計。
			layout : 左から 1, 2, 3
			place : コメント数表示の右側
		C3. コメントツールチップ
			image : comment_tooltip.png
			parts :
				1. コメント
				2. 日時
			layout : 左から 1, 2
			place : カーソル付近
	action
		A1. ニコる
			target : コメント欄のコメントのうち、今回の視聴でニコる操作をされていないコメント
			on : ダブルクリック
			effect :
				ニコるくん画像が反時計回りに 90 度回転し、グレーになる
				C1 のニコられ数が 1 増える
				コメントのスタイルが T1 の通りに更新される
				サーバーにニコる情報が送信される
					ニコる情報
						動画ID
						コメ番
		A2. 範囲ニコる
			target : コメント
			on : ドラッグ
			effect :
				ドラッグ範囲のコメントに対して、ニコる操作と同様の効果をもたらす
		A3. リロード
			target : コメントリロードボタン
			on : クリック
			effect :
				公式のコメントリロード処理と同時にサーバーからニコる情報を取得し、C2.3 を更新する。
		A4. ニコる表示
			target : コメント
			on : マウスオン
			effect :
				C1 を表示する
				C3 を表示する（公式機能）
		A5. ニコる消去
			target : コメント
			on : マウスアウト
			effect :
				C1 を消去する
				C3 を消去する（公式機能）
		A6. コメントスタイリング
			target : コメント
			on : コメント欄の中でコメントが表示された時
			effect :
				T1 の通りのスタイルをコメントに適用する
